# Tech Stack Icons

Add your custom tech stack icons here!

## Icon Format:
- **Type:** SVG (preferred) or PNG
- **Size:** 32x32px to 64x64px
- **Color:** Monochrome (will be tinted sepia)
- **Background:** Transparent

## Required Icons:

- **Simple Icons** - https://simpleicons.org/
